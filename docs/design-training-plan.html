Interactive Training Tutorial

2 Design the Training Plan

Before opening Unity, prepare the structure and content of your interactive training. Planning the training in advance will make it easier to configure the project template and reduce errors during development.

This chapter will guide you through the following tasks:

  • Selecting the video format
  • Preparing the questions and answer options
  • Writing instructions and feedback
  • Identifying when each element will appear
  • Opening the project template in Unity
  • Completing the planning documents

2.1 Step 1: Select the Video Format

Begin by deciding how the training video should respond to the participant’s answers.

There are two available formats.

2.1.1 Option 1: Continuous Video With Embedded Questions

In this format, the participant watches one continuous video.

At predetermined points:

  1. The video pauses.
  2. A question appears on the screen.
  3. The participant selects an answer.
  4. The video resumes.
  5. The process repeats when the next question is reached.

Recommended use: Choose this format when all participants should watch the same video sequence while answering questions that assess their understanding.

2.1.2 Option 2: Branching Video With Different Outcomes

In this format, the video sequence changes depending on the participant’s answer.

For example:

  • A correct answer may continue to the next part of the training.
  • An incorrect answer may show corrective feedback.
  • A selected answer may lead to a different video, scene, or outcome.

Recommended use: Choose this format when participants’ decisions should affect the storyline, setting, or sequence of the training.

To use a branching format, prepare all required materials in advance, including:

  • Each video segment
  • The instructions presented during each segment
  • The questions
  • The answer options
  • The correct answer for each question
  • The feedback presented after incorrect answers
  • The destination video or scene associated with each answer

2.1.3 Video Format Comparison

Feature Continuous Video Branching Video
Uses one primary video sequence Yes Not necessarily
Questions pause the video Yes Yes
All participants see the same sequence Yes No
Answers can change the storyline No Yes
Requires multiple prepared video segments No Usually
Complexity Lower Higher

2.2 Step 2: Prepare the Questions

Determine how many questions will appear in each training module.

For every question, prepare:

  1. The question text
  2. The possible answer options
  3. The correct answer
  4. The feedback associated with each answer

Current limitation: The project currently includes templates for questions with either two or four answer options.

Use the following format when planning each question:

Component Example
Question What should the caregiver do next?
Answer A Remove the hearing aid
Answer B Redirect the child to the activity
Correct answer Answer B
Correct-answer feedback Correct. Redirecting helps the child continue participating.
Incorrect-answer feedback Try again. Removing the hearing aid may reinforce the behavior.

2.3 Step 3: Prepare the Instructions

Write the instructions that participants will see before each question.

The instructions should tell participants what to observe or consider while watching the video.

For example:

Instruction: Watch how the caregiver responds when the child attempts to remove the hearing aid.

Keep each instruction:

  • Short
  • Clear
  • Directly related to the upcoming question
  • Visible long enough for the participant to read it

2.4 Step 4: Prepare the Feedback

Write the feedback that will appear after the participant selects an answer.

Prepare feedback for:

  • The correct answer
  • Each incorrect answer

2.4.1 Correct-Answer Feedback

Correct-answer feedback should:

  • Confirm that the selected answer was correct
  • Briefly explain why it was correct
  • Reinforce the important training concept

2.4.2 Incorrect-Answer Feedback

Incorrect-answer feedback should:

  • Indicate that the selected answer was incorrect
  • Explain why the answer was not appropriate
  • Redirect the participant toward the relevant instruction
  • Allow the participant to try again, when applicable

Tip: Feedback should teach the participant, not only indicate whether the answer was correct or incorrect.

2.5 Step 5: Identify When Each Element Will Appear

Determine the exact time at which each instruction and question should appear in the video.

For each item, record:

  • The instruction appearance time
  • The question appearance time
  • The video location associated with the item
  • What should happen after the participant answers

For example, an instruction may appear:

  • One minute before the related question
  • Immediately before a target behavior occurs
  • Immediately after a specific scenario is shown
  • At the beginning of a new training segment

2.5.1 Example Timeline

Video Time Element Content Action
02:00 Instruction Observe how the caregiver responds. Continue playing
03:00 Question 1 What should the caregiver do next? Pause video
05:00 Instruction Pay attention to the child’s communication. Continue playing
06:00 Question 2 Which response is most appropriate? Pause video

2.6 Step 6: Install Unity and Unity Hub

Before opening the project template:

  1. Download Unity Hub.
  2. Create or sign in to your Unity account.
  3. Install the required version of the Unity Editor.
  4. Confirm that Unity Hub opens correctly.

Refer to the Setup Programs chapter for detailed installation instructions.

2.7 Step 7: Save the Project Template

Download or copy the Unity project template to your computer.

Save it in a location that is:

  • Easy to find
  • Separate from the original compressed file
  • Included in your regular backup system

For example:

Documents/
└── Interactive Training Projects/
    └── Training Project Template/
